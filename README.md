# wp-swiss Badge Widget
wp-swiss.com ist ein Verzeichnis von WordPress-Seiten in der Schweiz. Falls du auf wp-swiss zurücklinken möchtest, hilft dir dieses einfache Widget.
## Anwendung
1. Einfach wp-swiss-badge.php in dein Plugin-Verzeichnis tun und unter Plugins im Dashboard aktivieren. 
2. Wähle unter "Design" - "Widgets" das Widget und setz es in den Widget-Bereich deiner Wahl. Fertig!
## Bemerkung
wp-swiss.com ist kostenlos und die Community aller WordPress-AnwenderInnen in der Schweiz kann Vorschläge machen, was das Verzeichnis alles machen soll.
